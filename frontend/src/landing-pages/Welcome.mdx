# Welcome to the Digital Library

This is a fully customizable landing page powered by **MDX** (Markdown + React).

You can use standard markdown syntax here:
- **Lists**
- *Italics*
- And even custom React components!

For example, we can use a custom info box right here in the landing page:

<div style={{ padding: '1.5rem', backgroundColor: 'var(--bg-highlight)', borderLeft: '4px solid var(--accent)', borderRadius: '4px', margin: '1.5rem 0' }}>
  ### Pro Tip
  You can edit this landing page directly in `frontend/src/landing-pages/Welcome.mdx` without writing complex HTML or TSX code!
</div>
