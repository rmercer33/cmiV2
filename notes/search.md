# Ideas About Search

* Search will look for exact match of the query but it would be helpful to do
  fuzzy matching where the query string is used as a suggestion. For example,
  I searched for 'beware of the temptation to see yourself unfairly treated'
  which was not found in ACIM but 'beware of the temptation to perceive
  yourself unfairly treated' is what is in the text.

  Search should be able to find that.

  Another example: I searched for 'hang up the phone and get on with it' was
  not found but 'hang up the telephone and get on with it' is what was present
  in WOM'.
